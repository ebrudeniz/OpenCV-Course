# 👁️ OpenCV Bootcamp - Practical Implementation

This repository contains my personal implementations, code snippets, and study notes from the official **OpenCV Bootcamp** (CV0) provided by [OpenCV.org](https://courses.opencv.org/). The project covers a comprehensive path from fundamental image processing to advanced deep learning-based detection models.

## 🚀 Course Curriculum

The contents are structured following the bootcamp modules:

* **Fundamentals:** Getting started with images, basic manipulation (resizing, cropping, color spaces).
* **Annotation & Enhancement:** Drawing shapes/text on images and applying enhancement techniques (brightness, contrast, etc.).
* **Video Processing:** Accessing camera feeds and writing processed frames to video files.
* **Image Processing:** Advanced filtering techniques, feature detection, and image alignment.
* **Computational Photography:** Creating panoramas (stitching) and High Dynamic Range (HDR) imaging.
* **Detection & Tracking:** Implementation of object tracking and face detection algorithms.
* **Deep Learning Integration:** * Object detection using the **TensorFlow** API.
    * Human pose estimation using **OpenPose**.
